# iOS Bindings

This folder contains iOS-facing artifacts for `platform-bridge`:

- `MobileAIRuntime.h`: C ABI header exposed by Rust.
- `module.modulemap`: Swift module map for the FFI header.
- `MobileAIRuntime.swift`: Swift wrapper over the C ABI.
- `MobileAIRuntime.xcframework`: generated by `scripts/build-ios.sh`.

## Build

From repo root:

```bash
scripts/build-ios.sh
```

Optional backend feature override:

```bash
FEATURES=llama-backend scripts/build-ios.sh
```

The script builds static libs for:

- `aarch64-apple-ios`
- `aarch64-apple-ios-sim`
- `x86_64-apple-ios`

Then it regenerates the header with `cbindgen` and packages an `.xcframework`.

## SwiftUI app wiring

The app scaffold lives at:

- `ios/MobileAIRuntimeApp/MobileAIRuntimeApp.xcodeproj`

Quick setup:

```bash
scripts/setup-ios-app.sh
```
